# react-webpack-boilerplate v 1.1

For jumping into new projects that need React | webpack | SASS

### Technologies used

- Babel
- Webpack
- Sass
- React


## Installation

```bash
Fork & clone the repo
Npm run install
```

## Scripts

```bash
NPM start - runs webpack in development with hot reload

Client code runs on localhost:8080

```
